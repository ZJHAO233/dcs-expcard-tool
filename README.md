```
convert(file_path)
├── 读取Excel文件
├── 遍历每一行 _process_row(row, rows, i)
│   │
│   ├── 空行/表头 → 跳过
│   │
│   ├── 二级标题（一、xxx）
│   │   ├── 输出空行
│   │   ├── 输出 '## xxx'
│   │   └── 重置状态
│   │
│   ├── 段落标题（试验条件/试验恢复/结论/存在问题）
│   │   ├── 试验条件 → skip_section = True
│   │   └── 其他 → in_content_area = False
│   │
│   ├── 试验内容 → in_content_area = True
│   │
│   ├── 三级标题（数字.中文）
│   │   ├── 统一括号为全角
│   │   ├── 输出空行
│   │   ├── 输出 '### xxx'
│   │   ├── 输出空行
│   │   └── 记录 current_sub_title_logic
│   │
│   ├── 子列一级（纯数字）_process_level1()
│   │   ├── 检查是否有子项（下一行是否是 x.1）
│   │   ├── 获取内容（跳过逻辑分隔符）
│   │   ├── 判断是否输出逻辑
│   │   │   ├── 有子项 + B列是逻辑分隔符
│   │   │   │   ├── B列逻辑 ≠ 三级标题逻辑 → 输出（B列逻辑）
│   │   │   │   └── B列逻辑 = 三级标题逻辑 → 忽略
│   │   │   ├── 有子项 + B列不是逻辑分隔符 → 使用子项B列逻辑
│   │   │   └── 无子项 → 不输出
│   │   ├── 处理或取反
│   │   └── 输出 '数字. 内容（逻辑）'
│   │
│   └── 子列二级（x.1, x.2...）_process_level2()
│       ├── 检查C列是否有逻辑分隔符
│       ├── 获取内容
│       │   ├── C列是逻辑分隔符 → 从D列开始找
│       │   └── C列不是逻辑分隔符 → 从B列开始找
│       │
│       ├── 有C列逻辑 → 开始分组
│       │   ├── group_items = [当前内容]
│       │   ├── continuation_logic = C列逻辑
│       │   ├── 循环检查后续行
│       │   │   ├── 同级子项（x.1, x.2...）
│       │   │   │   ├── 下一行C列有逻辑 → break（新分组）
│       │   │   │   └── 下一行C列无逻辑 → 收集内容
│       │   │   ├── 延续行（A列空，有内容）
│       │   │   │   ├── 下一行C列有逻辑 → break
│       │   │   │   └── 收集内容
│       │   │   └── 其他 → break
│       │   │
│       │   └── 输出分组结果
│       │       ├── 多项 → '   - 条件1 且/或 条件2'
│       │       └── 一项 → '   - 条件1'
│       │
│       └── 无C列逻辑 → '   - 内容'
│
└── 返回 '\n'.join(output)
```
